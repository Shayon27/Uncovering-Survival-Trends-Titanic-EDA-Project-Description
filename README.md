# Uncovering-Survival-Trends-Titanic-EDA-Project-Description
his project explores the famous Titanic dataset to uncover patterns in passenger survival. It demonstrates the end-to-end data analysis process, starting with data ingestion and robust cleaning (handling null values and outliers) to detailed visualization
Key Features:

Data Cleaning: Robust handling of missing Age and Embarked data.

Visualization: Insightful charts including survival heatmaps, age distribution histograms, and categorical breakdowns by gender and class.

Tech Stack: Python, Pandas, NumPy, Matplotlib, Seaborn.
